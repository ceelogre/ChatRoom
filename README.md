# fseChatRoom
Chat room app for FSE class.  
To start the backend server, `cd` into backend and run  
` npm i `   
then   
`nodemon`  


To run the client, `cd` into frontend and run  
`npm i`  
then  
`npm run serve  `
or  
`npm run build`  
for a production ready app.  
Browse over to `localhost:8080` and Voila!
